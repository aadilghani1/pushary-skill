# Pushary Agent Skill

Push notifications and human-in-the-loop for AI coding agents.

## Install

```bash
npx skills add pushary/pushary-skill
```

Works with Cursor, Claude Code, Windsurf, Codex, and [39 more agents](https://skills.sh/).

## What it does

- **Task alerts** — your agent sends a push notification to your phone when a task finishes or fails
- **Human-in-the-loop** — your agent asks a yes/no question via push, you answer from your lock screen, the agent continues
- **MCP native** — connects via the Model Context Protocol, 4 lines of JSON config
